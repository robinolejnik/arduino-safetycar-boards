# arduino-safetycar-boards

## Installation

Add `https://github.com/robinolejnik/arduino-safetycar-boards/releases/latest/download/package_safetycar_index.json` to the additional board manager URLs in the Arduino IDE.

The list can be found under `File -> Preferences -> Additional board manager URLs`.

Go to the Boards Manager and search for `SafetyCar Boards`. Click install.
